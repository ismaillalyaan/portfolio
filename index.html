AI-Powered Academic Advisor Chatbot

Production-grade academic advisor chatbot for a university, deployed as a REST API (FastAPI + Uvicorn) integrated into a university mobile app, serving students across 3 program tracks. Project 02
Flutter Groq OpenCV
Nestora — AI Floor Plan Furnishing App

Full-stack Flutter application (Provider state management, Supabase auth/database/storage) that takes a room floor plan photo through a multi-step design wizard — dimensions, door/window placement, style, furniture preferences — and returns an AI-furnished 2D CAD-style layout. The FastAPI AI backend uses a Groq vision model (Llama-3.2-11B-Vision) to detect room labels, OpenCV flood-fill to segment room polygons from a single seed point, and a Groq LLM (Llama-3.3-70B) to plan furniture placement rendered as color-coded CAD symbols. A Supabase-backed workshop-request flow closes the loop from AI design to a real furniture order.

Project 03
RAG Pinecone Bilingual
BNU Admission Chatbot — Bilingual RAG Assistant

Bilingual (Arabic/English) RAG chatbot answering prospective-student admission questions for Benha National University, served through a FastAPI backend with a lightweight web frontend. Hybrid retrieval combines a Pinecone dense vector index (bge-m3 embeddings via the HuggingFace Inference API) with local BM25 sparse search fused via Reciprocal Rank Fusion, plus an optional Cohere reranker.
